How to use Tabber ???
1) To go to any tab using its 'Serial Number' just enter the number and clck NUMBER

2) To go to any tab using its 'Tab Name' just enter the name and click NAME

3) To rename any tab just enter the new name and click RENAME

4) To reset custom names given to all previous tabs, click ALL under RESET heading

5) To reset name of only current tab, click CURRENT under RESET heading

6) You can see the current tab under NOW heading

7) If the D is in green then clicking on any number will navigate you to that tab

8) If the D is in red then clicking on any number will delete that tab
